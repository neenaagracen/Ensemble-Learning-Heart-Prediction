# Heart Disease Prediction using Ensemble Learning

This project predicts heart disease using Random Forest (an Ensemble Learning technique).

## Features
- Data preprocessing using Label Encoding
- Model comparison (Logistic Regression, Decision Tree, Random Forest)
- Random Forest selected as best model
- Streamlit web application for prediction

## Files
- Ensemble_Learning_Techniques.ipynb → Model training
- app.py → Streamlit application
- model.pkl → Saved model
- heart.csv → Dataset
- screenshot.png → Output preview

## Run the App
streamlit run app.py

## Result
The model predicts whether a person has heart disease based on input features.
